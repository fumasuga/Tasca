Tasca プライバシーポリシー

1. はじめに

Tasca（以下「本アプリ」といいます）は、ユーザーの個人情報の保護を重要視しています。本プライバシーポリシーは、本アプリがどのような個人情報を収集し、どのように使用・保護するかを説明するものです。

2. 収集する情報

本アプリは、以下の情報を収集します：

  • アカウント情報：メールアドレス、パスワード（暗号化して保存）
  • タスク情報：作成したタスク、完了したタスク、タスクの内容、アウトプット、URLリンク
  • 使用状況情報：アプリの使用状況、エラー情報
  • デバイス情報：デバイスの種類、OSバージョン、一意のデバイス識別子

3. 情報の利用目的

収集した情報は、以下の目的で利用します：

  • 本アプリの提供、維持、改善
  • ユーザーアカウントの管理と認証
  • タスクデータの保存と同期
  • お問い合わせへの対応
  • セキュリティの確保と不正利用の防止
  • サービスの改善のための分析

4. 情報の保存とセキュリティ

  1. 本アプリは、Supabase（PostgreSQLデータベース）を使用してデータを保存します。
  2. すべてのデータはRow Level Security (RLS)により保護され、ユーザーは自分のデータのみアクセスできます。
  3. パスワードは暗号化（ハッシュ化）して保存され、平文で保存されることはありません。
  4. データベースへの通信はHTTPS（SSL/TLS）で暗号化されています。

5. 第三者への情報提供

当社は、以下の場合を除き、ユーザーの個人情報を第三者に提供することはありません：

  • ユーザーの同意がある場合
  • 法令に基づく場合
  • 人の生命、身体または財産の保護のために必要がある場合
  • 公衆衛生の向上または児童の健全な育成の推進のために特に必要がある場合

6. データの保存期間

ユーザーのデータは、アカウントが削除されるまで保存されます。アカウント削除時には、関連するすべてのデータが削除されます。

7. ユーザーの権利

ユーザーは、以下の権利を有します：

  • 自己の個人情報へのアクセス権
  • 自己の個人情報の訂正・削除を求める権利
  • アカウントの削除を求める権利
  • データのエクスポートを求める権利

これらの権利を行使する場合は、お問い合わせフォームよりご連絡ください。

8. クッキー（Cookie）

本アプリは、認証状態を維持するためにセッション情報を使用しますが、追跡目的のクッキーは使用しません。

9. 外部サービス

本アプリは、以下の外部サービスを使用しています：

  • Supabase：データベースと認証サービスの提供（プライバシーポリシー：https://supabase.com/privacy）

10. お子様の個人情報

本アプリは、13歳未満の子供を対象としていません。13歳未満の子供の個人情報を意図的に収集することはありません。

11. プライバシーポリシーの変更

当社は、必要に応じて本プライバシーポリシーを変更することがあります。変更があった場合は、本ページに掲載し、重要な変更についてはユーザーに通知します。

12. お問い合わせ

プライバシーポリシーに関するご質問やご意見がございましたら、お問い合わせフォームよりご連絡ください。
